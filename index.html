Projects

3D Video Game

A 3D space exploration and colonization game. I use sparse voxel octrees to allow me to render voxel planets made up of trillions of voxels generated procedurally by a compute shader. The surface is then extracted using the marching cubes algorithm, also in a compute shader.

Pneumatic Emergency Braking System

Pneumatic over hydraulic booster cylinder for the emergency braking system on the formula SAE car.
Because the car needs two booster cylinders of varying dimensions, I developed this project as one file that is fully parameter driven so I can easily generate mutliple versions as needed

Satellite receiving station

Used a software defined radio with a custom antenna and SDR# software to record analog image signals from NOAA weather sattelites. The recorded audio could then be decoded into images like the one above.

Mouse Camera

Some mice have serial debug ports left active from the factory. I then wrote a C# windows forms application to connect to the arduino over USB and display the image data as a live stream
